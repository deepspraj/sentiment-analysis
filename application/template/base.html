Services


Import Live Tweets

You can import live tweets by using targets twitter handle and last 10 tweets will perform sentiment analysis.

Import Live Hashtags

You can import live top 5 trending hashtag's and every hashtag's 5 latest tweet will perform sentiment analysis.

Type Your Own Tweet

You can enter your own tweet in raw format and perform sentiment analysis

Audio as Input

You can record audio here and perform sentiment analysis.

Upload Your Audio

You can upload audio (not more than 10 sec) and perform sentiment analysis.
